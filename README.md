# Cypress

## 1. Install

```npx cypress install```

### 1.1. cypress types with this command: 

```npm i -D @types/cypress@latest```

## 2. Run tests

### 2.1. Cypress Studio

```npx cypress open```

### 2.2. headless mode 

- all
```npx cypress run```

- specific file
```npx cypress run --spec "cypress/e2e/navigation.cy.js"```
